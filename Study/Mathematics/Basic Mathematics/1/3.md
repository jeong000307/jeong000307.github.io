## Rules for Multiplication

### 개념 요약

- 교환 법칙
    + $a$, $b$가 정수일 때, $a + b = b + a$ 이다.
    + (예시) $3 + 5 = 5 + 3 = 8$

- 결합 법칙
    + $a$, $b$, $c$가 정수일 때, $(a + b) + c = a + (b + c)$ 이다.
    + (예시) $(3 + 5) + 9 = 8 + 9 = 3 + (5 + 9) = 3 + 14 = 17$
